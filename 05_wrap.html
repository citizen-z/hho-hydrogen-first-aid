The array need to be wrapped to prevent unwanted current between the tips of the plates. Our target is keep the current between the perpendicular surfaces and prevent the rest.


It is possible to wrap the array with any insulated material as long it will stay in place in water. In my opinion it is best to wrap it using duct tape, but using shrink-wrap is also good idea.

When using duct tape make sure the PVC surface is clean and dry. As you wrap, keep tension on the tape so that it is pulled into the plates (see top view in drawing).



This is important step because the leakage current is causing the water to warm up and we want to prevent unnecessary energy losses like this.
